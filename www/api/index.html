GET
/api/municipality

Get list of available Municipalities within Victoria as defined by the Victorian Building Authority.
Download Victorian Building Authority Supplementary Information.

Parameter Value Type Description
None None None None

GET
/api/municipality/list/:municipality_id

Get a list of permits issued inside of a specified Municipality area.

Parameter Value Type Description
municipality_id integer Required The Municipality ID to search
year integer Specify the given year to search
month integer Specify the given month to search
type integer The type of work being completed
offset integer The number of rows to offset the results
limit integer Specify to the limit the number of items returned from the search

GET
/api/municipality/summary/:municipality_id

Get a summary of information for a specified Municipality area.

Parameter Value Type Description
municipality_id integer Required The Municipality ID to search
year integer Specify the given year to search
month integer Specify the given month to search

GET
/api/postcode/:postcode

Get a list of permits issued from a given postcode.

Parameter Value Type Description
postcode integer Required The postcode to search
year integer The year to search
month integer The month to search

GET
/api/suburb/:suburb

Get a summary of information for a specified Municipality area.

Parameter Value Type Description
suburb string Required Specify the given Suburb to search (buggy)
year integer Specify the given year to search
month integer Specify the given month to search
